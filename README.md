# Oakhurst Airbnb Investment Report

Interactive report analyzing for-sale homes in Oakhurst, CA (near Yosemite) as short-term-rental (Airbnb) investments.

- **`index.html`** — open in any browser (self-contained; photos stream from Redfin CDN).
- 30 homes (3+ bd, $300K–$800K, within ~12 min of downtown Oakhurst).
- Metrics: projected revenue, cash-on-cash, cap rate, ROI, DSCR (via bnbcalc), hot tub / HOA / STR-permit checks (top picks), Yosemite distance.

Sources: Zillow (listings), bnbcalc (projections), Redfin (listing details), Madera County (STR ordinance).
